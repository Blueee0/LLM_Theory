# Stanford CS229 机器学习课程学习笔记

这是我的 Stanford CS229（机器学习）课程的个人学习笔记仓库。此仓库用于记录学习过程中的重要概念、理解和实践。

## 课程简介

Stanford CS229 是斯坦福大学的核心机器学习课程，由 Andrew Ng 教授主讲。本课程涵盖以下主要内容：
- 监督学习算法与应用
- 无监督学习方法
- 深度学习基础
- 学习理论
- 机器学习项目实践
