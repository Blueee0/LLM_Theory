# Stanford CS336 语言建模课程学习笔记

这是我的 Stanford CS336（Language Modeling from Scratch）课程的个人学习笔记仓库。此仓库用于记录学习过程中的重要概念、理解和实践。

## 课程简介

CS336 是斯坦福大学的语言建模课程，主要关注从零开始构建语言模型的全过程。本课程涵盖以下主要内容：
- 分词与预处理技术
- Transformer模型架构设计
- 大规模训练系统优化
- 数据处理与清洗
- 模型对齐与安全
